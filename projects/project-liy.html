Project Summary

Lead-it-Yourself! is a National Science Foundation supported project led by Eve Riskin, and Jan Spyridakis of the University of Washington. The project aims to support schools across the nation in leading professional development for faculty to reduce biases in Engineering fields. I had the privellage of contributing to this project by leading the user experience design, which included user research, usability studies, content strategy, and web design.

Problem Space

How might we develop a website to support schools in leading workshops aimed at recognicing and reducing biases in Engineering fields?

Process

Before any development began, I conducted numerous interviews to fully understand the space, and develop a persona. Then as a team, guided by the user research I established a set of goals and tasks for the website.

I then created and iterated on an Axure prototype, guided by remotely-conducted usability studies. After many iterations, and when I stopped finding significant issues in the usability students we began development using a Drupal framework.

Results

The LiY! website is still in production, and will be released in Beta starting in Winter 2017.
